# WorldTree 架构插件用户指南
## 1. 简介
WorldTree是OpenClaw生态下的六层架构可视化与管理插件，支撑分布式Agent系统的全生命周期管理。
## 2. 安装
1. 打开OpenClaw SkillHub
2. 搜索`worldtree`
3. 点击安装，自动完成依赖部署
## 3. 快速开始
1. 执行`/worldtree init`初始化架构项目
2. 执行`/worldtree visualize`生成架构图
3. 执行`/worldtree validate`校验架构合规性
## 4. 核心功能
- 六层架构可视化展示
- 依赖关系自动梳理
- 架构合规性校验
- 多项目架构对比
## 5. 常见问题
Q: 初始化失败？A: 检查是否有写入权限，确保网络正常可拉取依赖。
