# API 接口契约

## 版本信息
- 版本: v1.0.0
- 基础路径: /api/worldtree/v1

## 通用规范
- 请求格式: JSON
- 响应格式: JSON
- 认证方式: OpenClaw JWT Token

## 通用响应结构
```json
{
  "code": 0,
  "msg": "success",
  "data": {}
}
```

## 错误码定义
| 错误码 | 说明 |
|--------|------|
| 0 | 成功 |
| 10001 | 参数错误 |
| 10002 | 权限不足 |
| 10003 | 资源不存在 |
| 10004 | 内部服务错误 |

## 核心接口列表
1. 节点管理接口: /node/*
2. 知识图谱接口: /kg/*
3. Agent协作接口: /agent/*
4. 权限管控接口: /perm/*
5. 资源调度接口: /resource/*

每个接口的详细请求参数、响应示例见对应接口文档。
